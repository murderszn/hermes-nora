// BRAIN INTEGRATION

Pollinations AI out of the box

Nora's app-side reasoning runs on Pollinations — one API key, multimodal kimi by default, tools and vision included. No provider juggling for new users; swap models in chat when you need to.

default.model kimi · 256K
provider custom:pollinations
setup onboard.sh writes .env
vision + tools native on kimi
