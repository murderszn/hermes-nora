Nora ships as a drop-in Hermes persona today. This page tracks what we're building next — voice interfaces, connector packages, and the skills that turn her into a full agent stack.

What you get today

Nora is live as a Hermes agent addon: persona, ops skills, channel gateways, persistent memory, subagent delegation, and MIT-licensed source on GitHub. Install with one command and reach her from Discord, email, Slack, and more.

v0 · persona

Hermes agent addon

Drop-in persona with ops dashboards, terminal access, and curated skills.

v0 · channels

Multi-channel gateway

Discord, Gmail, Slack, Telegram, WhatsApp, Signal, and CLI — wired through Hermes.

v0 · ops

Always-on monitoring

Natural-language scheduling for health checks, backups, and incident briefings.

Planned

Voice

Talk to Nora out loud. We're adding speech-to-text and text-to-speech pipelines so you can reach her hands-free — from a desk mic, phone, or smart speaker hookup.

TTS & STT setup guides

Step-by-step configuration docs are coming for:

  • Text-to-speech (TTS) — local and cloud providers, voice selection, latency tuning
  • Speech-to-text (STT) — wake-word options, streaming transcription, push-to-talk
  • Gateway integration — routing voice sessions through the Hermes channel layer
  • Hardware notes — mic arrays, edge devices, and low-latency audio paths
planned

STT pipeline

planned

TTS responses

Nora speaks back through your chosen voice engine — with persona-aware tone and pacing.

planned

Setup documentation

Full install and config walkthroughs for TTS/STT — same quality as our channel docs.

Planned

Connector packages

Modular skill packs that extend Nora for specific workflows. Each connector is a self-contained package — install only what you need, mix and match across machines.

Marketer

Campaign briefs, copy drafts, and analytics summaries.

Coder

Repo-aware dev workflows, PR review, and deploy scripts.

Assistant

Calendar, inbox triage, and daily briefings.

Designer

Layout feedback, design tokens, and asset handoff.

Content editor

Image, video, and long-form content pipelines.

Therapist

Reflective journaling and guided check-ins — local and private.

Document repo

Indexed knowledge base with semantic search and citations.

Media server

Library management, transcoding, and streaming hooks.

Package model

Connectors ship as installable packages with their own skills, configs, and optional dashboards. They plug into Nora's existing delegation and memory layers — no fork required. Want one we haven't listed? Open an issue or bring it to Discord.

Shape the roadmap

Nora is open source. Priority follows what the community actually deploys — vote with installs, issues, and PRs. Join Discord to propose connectors or help write the voice setup guides.
